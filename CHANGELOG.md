# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-15

### Added

- `<bsky-comments>` Web Component with Light DOM rendering.
- Dual input modes: public Bluesky URL (`post`) or AT-URI (`uri`).
- Automatic handle-to-DID resolution for public URLs.
- Nested reply threading with configurable sort order (`asc` / `desc`).
- Customizable Like and Reply icons via attributes or CSS.
- Configurable PDS endpoint via `service` attribute.
- TypeScript type declarations including JSX `IntrinsicElements` support.
- Zero runtime dependencies.
